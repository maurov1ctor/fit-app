# Fitness App

Aplicação fitness com acompanhamento de alunos por professores.

## 🚧 Status do Projeto

Atualmente em desenvolvimento.

✔ Backend (NestJS) — em progresso  
⏳ Frontend — ainda não iniciado  
⏳ Mobile — planejado  

Fase atual: Autenticação (JWT + Refresh Token)

## 🚀 Tecnologias
- React
- NestJS
- PostgreSQL
- Prisma

## 🧠 Arquitetura

Projeto baseado em:
- Clean Architecture
- Modularização com NestJS
- Prisma ORM
- Autenticação JWT com Refresh Token

## 📦 Estrutura atual

backend/

## 🔐 Funcionalidades
- Autenticação JWT
- Controle de treinos
- Monitoramento de progresso

### Backend
cd backend
npm install
npm run start:dev

### Frontend
cd frontend
npm install
